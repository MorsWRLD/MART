Connect to SoundCloud

1. Open soundcloud.com and make sure you're logged in
2. Press F12 to open DevTools, go to Console
3. Type allow pasting and press Enter
4. Paste this and press Enter:
document.cookie.match(/oauth_token=([^;]+)/)?.[1]
5. Copy the result (the string in quotes) and paste in the first field below
6. Then run document.cookie and copy the full result into the second field
(The full cookies are needed for playlist creation — SoundCloud blocks server-side writes without them)
